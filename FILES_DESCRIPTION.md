# Danh sách file còn lại trong thư mục

## File chính
- `realtime_monitor_optimized.js` - Script chính mới nhất với tất cả tính năng đã cập nhật

## File cấu hình
- `ecosystem.config.js` - File cấu hình PM2 để quản lý tiến trình

## File demo và test (giữ lại để tham khảo)
- `demo_new_format.js` - Script demo định dạng số mới (có dấu chấm từ 1 triệu trở lên)
- `demo_notification.js` - Script demo định dạng thông báo
- `mock_test.js` - Script mô phỏng kiểm tra định dạng thông báo thực tế
- `notification_tester.js` - Script kiểm tra xác nhận chỉ có một bảng thông báo duy nhất
- `test_new_format.js` - Script test định dạng số mới
- `test_notification_format.js` - Script thử nghiệm định dạng thông báo
